## 📄 Overview

<a href="#">PixelSnap</a> fills the specific gap in the design-to-development workflow where precise measurements are needed from content on screen rather than from design source files — checking that a live web page matches a design specification, measuring spacing in a competitor's interface, or verifying that implemented UI matches intended dimensions.

The <a href="#">screenshot with measurement overlay</a> captures the current measurement and produces an annotated screenshot with dimensions marked for documentation and sharing with development teams. <a href="#">Measurement history</a> records recent measurements for reference without requiring re-measurement of previously measured values.

<p align="center">
  <img src="https://pixelsnap.com/img/og-image.png" alt="PixelSnap screenshot"/>
</p>

<a href="#">Integration with design tools</a> works alongside Figma, Sketch, and other design applications — measuring content inside these applications as naturally as measuring any other screen content. <a href="#">Customizable keyboard shortcuts</a> configure the activation key combination for integration with existing workflow shortcuts without conflicts.

The <a href="#">multi-display support</a> measures content on any connected display with correct pixel values for each display's resolution and density. <a href="#">Ruler tool mode</a> provides a traditional ruler overlay for measuring distances along a single axis when precise linear measurement is needed.

The smart edge detection is what makes PixelSnap practically faster than manual pixel-level measurement rather than just conceptually faster. Precisely positioning a measurement handle at the exact pixel boundary of a design element requires careful cursor placement that slows the measurement significantly. Smart edge detection identifies the boundaries of visual elements and snaps to them automatically, which means measurements complete in one gesture rather than requiring precise cursor positioning at exact pixel coordinates. The snap accuracy also reduces measurement error.
